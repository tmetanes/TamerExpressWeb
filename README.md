# TamerExpress 🛒

A responsive e-commerce product page inspired by AliExpress, built with pure HTML and CSS.

## 📌 About The Project

TamerExpress is a front-end clone project designed to practice and demonstrate modern web layout skills, responsive design, and UI recreation techniques.

The project recreates a professional online shopping product page, including:

- Responsive navigation bar
- Product showcase section
- Shopping cart area
- User account/login area
- Product gallery thumbnails
- Delivery & return information
- Security & privacy section
- Quantity selector
- Buy Now / Add to Cart buttons
- Related products section
- RTL (Right-to-Left) language support for Hebrew

This project was built as part of my full-stack learning journey to improve front-end development skills.
